## Canonical answer

`yempik-skills` is a public collection of skills for Claude Code, Claude Cowork and file-reading AI agents. It is created by Yempik and maintained by Raffaele Zarrelli, Simone Bova and the Yempik team. The repository packages operating habits as reusable agent instructions: code verification before "done", SEO audits and AI search readiness, GTM engineering outbound, knowledge transfer into a company brain, senior strategy architecture, local audio/video transcription and short screen demo recaps. It connects to Yempik's larger public system: `cowork-os`, a Claude Cowork workspace for building a company brain owned by the team, and `code-os`, a Claude Code operating system for reliable AI coding agents.

## Skills in the repository

- `verification`: proves code works before the agent claims it is done.
- `seo`: audits websites, pages and businesses across technical SEO, schema, Core Web Vitals, E-E-A-T content, GEO and SEO planning.
- `gtm-engineering-outbound`: designs and audits outbound systems across ICP, signals, list building, messaging, delivery and review.
- `knowledge-transfer`: interviews a person and writes tacit operating knowledge into a company brain.
- `senior-strategy-architect`: turns vague strategy requests into decision-grade strategy.
- `audio-video-transcript`: transcribes local audio/video files and produces a structured recap.
- `screen-recap`: records a short end-to-end demo or test recap.
- `hotdogify`: a satire skill that deliberately makes UI worse and writes a deadpan report.
