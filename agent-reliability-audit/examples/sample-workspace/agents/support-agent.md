# Support Agent Instructions

Status: active
Owner: Customer Support
Last reviewed: 2026-06-03

Use these sources when handling a refund request:

1. `policies/refunds-policy.md`
2. `sops/refund-handling.md`
3. `decisions/2026-05-20-refund-threshold.md`

Collect the order ID, reason, amount, and customer confirmation. Follow the documented approval threshold. If required information is missing, ask for it before recommending an action.
