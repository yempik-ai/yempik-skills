# Framework Library

Use frameworks as thinking tools, not as decoration. Name a framework only when it helps the user understand the decision.

## Core strategy

### Strategy kernel
Use for all strategy tasks. Output: diagnosis, guiding policy, coherent actions.
Common misuse: replacing diagnosis with a goal.

### Strategy choice cascade
Use for company, business unit, GTM, product, or competitive direction.
Output: winning aspiration, where to play, how to win, capabilities, management systems.
Common misuse: answering every box with vague ambitions.

### Five Forces
Use for industry attractiveness, competitive pressure, pricing power, threats, and market entry.
Output: force-by-force implications and strategic posture.
Common misuse: treating it as a generic competitor list.

### Value chain
Use for margin, operations, differentiation, cost advantage, service model, or capability strategy.
Output: activities that create advantage, cost, risk, or customer value.
Common misuse: listing activities without identifying where advantage is created.

### BCG growth-share matrix / portfolio logic
Use for portfolio allocation across products, segments, business lines, or channels.
Output: invest, harvest, fix, exit, or test decisions.
Common misuse: using it without market share/growth evidence.

### Three horizons
Use for growth portfolios: current engine, near-term expansion, future bets.
Output: Horizon 1/2/3 initiatives with different metrics and governance.
Common misuse: treating horizons as sequential phases instead of parallel portfolios.

## Customer, value, and positioning

### Jobs to Be Done
Use for customer insight, product strategy, messaging, innovation, and positioning.
Output: functional, emotional, and social job; triggers; obstacles; desired progress.
Common misuse: writing vague personas instead of buying/usage situations.

### Value Proposition Canvas
Use for matching customer jobs, pains, gains with product pain relievers and gain creators.
Output: value proposition fit and proof gaps.
Common misuse: filling boxes without customer evidence.

### STP: segmentation, targeting, positioning
Use for marketing, GTM, brand, and channel strategy.
Output: chosen segment, target priority, position.
Common misuse: “everyone” as target.

### Positioning statement
Use for brand/product/category clarity.
Output: target, frame of reference, point of difference, reasons to believe, points of parity.
Common misuse: making a claim with no proof.

### Elements of Value
Use for B2B/B2C value proposition, sales messaging, pricing power, and differentiation.
Output: value drivers by segment and proof needed.
Common misuse: assuming every value element matters equally.

## Growth and product

### AARRR funnel
Use for diagnosing acquisition, activation, retention, referral, revenue.
Output: bottleneck by stage and experiments.
Common misuse: optimizing top-of-funnel while retention is broken.

### Growth loops
Use for products/channels where outputs can feed new inputs: referrals, content, SEO, collaboration, marketplaces, data, community.
Output: loop diagram in words: input → action → output → reinvestment.
Common misuse: calling any repeated tactic a loop.

### North Star Framework
Use for product/growth alignment.
Output: North Star metric, input metrics, guardrails, and initiatives.
Common misuse: choosing revenue only when it does not reflect customer value.

### Lean Build-Measure-Learn
Use for uncertainty, MVPs, experiments, PMF, and early-stage strategy.
Output: hypothesis, MVP/test, metric, threshold, decision.
Common misuse: building before defining the learning goal.

### RICE / ICE prioritization
Use for roadmap, growth experiments, content bets, or channel tests.
Output: ranked bets with reach/impact/confidence/effort or impact/confidence/ease.
Common misuse: false precision with made-up scores.

### OKRs
Use to turn strategy into execution.
Output: qualitative objective + 2-5 measurable key results.
Common misuse: making tasks into KRs instead of outcomes.

## Marketing, sales, and revenue

### Full-funnel measurement
Use for media, brand/performance balance, lifecycle, and attribution limits.
Output: stage metrics, incrementality/experimentation plan, business KPI linkage.
Common misuse: judging brand activity only by last-click conversions.

### Buyer journey mapping
Use for B2B sales/marketing alignment, complex decisions, and content/sales enablement.
Output: buying jobs, stakeholders, friction, assets, next-best actions.
Common misuse: assuming a linear journey.

### Sales pipeline stages
Use for sales process, forecasting, and conversion diagnosis.
Output: stage definitions, exit criteria, conversion metrics, coaching priorities.
Common misuse: stages based on seller activity rather than buyer commitment.

### Value-based pricing
Use when value can be quantified or segmented.
Output: customer value, next-best alternative, willingness-to-pay hypothesis, capture share.
Common misuse: ignoring customer proof, competitive anchors, or cost floor.

### Pricing metric and packaging
Use for SaaS, services, marketplaces, subscriptions, usage products, and tiering.
Output: value metric, tiers, fences, add-ons, discount rules, test plan.
Common misuse: charging per seat when value scales with usage/outcomes.

### NRR / retention economics
Use for subscription/customer success strategy.
Output: churn, contraction, expansion, adoption, health score, account plan.
Common misuse: treating retention as customer support instead of product/value realization.

## Innovation, competition, and market creation

### Blue Ocean / Four Actions / ERRC
Use to break category sameness or redesign value curves.
Output: eliminate, reduce, raise, create.
Common misuse: pretending there is no competition when substitutes exist.

### Moat mapping
Use for competitive advantage.
Output: brand, network effects, switching costs, data, cost advantage, distribution, regulatory/capability moats.
Common misuse: calling a feature a moat.

### Pre-mortem
Use for high-risk strategies.
Output: top failure modes and mitigations.
Common misuse: listing risks without changing the plan.

## Fundraising, marketplace, and community

### Investor narrative spine
Use for fundraising and pitch decks.
Output: problem, insight, solution, why now, market, traction, model, team, ask, milestones.
Common misuse: describing the product instead of creating belief in a venture-scale outcome.

### Marketplace liquidity
Use for marketplaces/platforms.
Output: atomic network, supply/demand constraint, liquidity metric, cold-start plan.
Common misuse: chasing total users instead of successful matches.

### Community business-value model
Use for community strategy.
Output: business objective, member value, behaviors, rituals, governance, metrics.
Common misuse: measuring posts/members without linking to business outcomes.

### Partnership ecosystem model
Use for partner-led growth.
Output: partner type, mutual value, incentive, co-sell/referral/integration motion, partner-sourced/influenced metrics.
Common misuse: collecting logos without revenue or customer value.
